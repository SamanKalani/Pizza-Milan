# 🍕 Pizza Milan

A modern, fast, and fully responsive online food ordering application built with React.js. This project features a tailored crimson-red and charcoal-black design system, moving away from standard templates to deliver a premium, high-end user experience.

Built as a key portfolio project to demonstrate advanced state management, custom routing architectures, and modern utility-first styling.

## 🔗 Live Demo

The application is fully deployed and production-ready on Vercel:
👉 **[pizza-milan-tau.vercel.app](https://pizza-milan-tau.vercel.app)**

---

## ✨ Key Features

- **Advanced State Management:** Architected using **Redux Toolkit (RTK)** to manage complex global states including shopping cart dynamics, user authentication, and real-time order tracking.
- **Modern Routing (React Router 6.4+):** Utilizes powerful new data-fetching paradigms such as `loader` functions to pre-fetch menu data before rendering, and `action` functions for seamless form submission.
- **Custom UI/UX & Tailwind Design:** Styled with a highly customized **Tailwind CSS** configuration, featuring a dynamic crimson-red theme (`#C40707`), sleek charcoal-black cart overviews, and subtle amber accents.
- **Real-Time Order Tracking:** Simulates live kitchen and delivery statuses, allowing users to track their orders via a unique order ID.
- **Geolocation Integration:** Fetching user location automatically using the browser's Geolocation API to streamline the delivery address input.

---

## 🛠 Tech Stack

- **React.js** (Core Library - Functional Components & Hooks)
- **Redux Toolkit** (Global State & Cart Architecture)
- **React Router (v6.4+)** (Loaders, Actions, and Declarative Routing)
- **Tailwind CSS** (Custom Production Design System)
- **Vite** (Next-Generation Frontend Tooling)

## 🚀 Local Installation & Setup

To run this project locally on your machine, follow these steps:

1. Clone the repository:

```bash
git clone [https://github.com/SamanKalani/pizza-milan.git](https://github.com/SamanKalani/pizza-milan.git)
```

2. Navigate into the project folder:
   cd pizza-milan

3. Install the required project dependencies:
   npm install

4. Start the Vite development server:
   npm run dev

Open your browser and navigate to http://localhost:5173 to explore the app!
